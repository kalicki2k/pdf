Dieses Paket enthaelt kleine gemeinsame Stil-Primitiven fuer den internen PDF-Kern.

- `Color` beschreibt PDF-Farbraeume und rendert die passenden Stroking-/Non-Stroking-Operatoren.
- `Opacity` beschreibt gemeinsame Fill-/Stroke-Opacity und rendert das ExtGState-Dictionary.

Die Typen sind bewusst generisch und werden von Text-, Tabellen-, Formular- und Seiten-Content-Pfaden gemeinsam verwendet.
