Dieses Paket enthaelt deklarative Styles fuer seitennahe Komponenten.

- `BadgeStyle` beschreibt Padding, Farben, Border und Opacity fuer Badges.
- `PanelStyle` beschreibt Layout- und Stilparameter fuer Panels.
- `CalloutStyle` erweitert `PanelStyle` um Pointer-spezifische Stilwerte.

Die Typen liegen bewusst nahe an `PageComponents`, weil sie nur fuer diese High-Level-Komponenten relevant sind.
