# Layout Table

Dieses Paket enthaelt den Tabellenkern.

- `Definition` enthaelt deklarative Eingabetypen wie `TableCell`, `TableCaption` und `TableHeaderScope`.
- `Style` enthaelt deklarative Stilobjekte wie `TableStyle`, `RowStyle`, `HeaderStyle`, `FooterStyle`, `CellStyle`, `TableBorder` und `TablePadding`.
- `Layout` enthaelt Zeilen- und Zellvorbereitung.
- `Rendering` enthaelt den mehrseitigen Tabellen-Renderpfad.
- `Support` enthaelt Stilauflosung und tabellenspezifische Hilfsobjekte.
