Dieses Paket enthaelt die dokumentweite TOC-Konfiguration.

- `TableOfContentsOptions` buendelt Titel, Font, Margin, Stil und Platzierung.
- `TableOfContentsPlacement` und `TableOfContentsPosition` beschreiben die Einfuegestelle.
- `TableOfContentsStyle` und `TableOfContentsLeaderStyle` beschreiben die Darstellungsregeln.
